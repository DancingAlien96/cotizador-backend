# cotizador-backend

Backend del sistema **Cotizador PROMESA**.

El frontend (Next.js) vive en un repositorio aparte y actualmente maneja los
datos en archivos JSON locales. Este backend se construirá para centralizar:

- Autenticación / usuarios
- Persistencia de cotizaciones (tienda, Guatecompras, empresas) en base de datos
- Numeración de folios
- API para el frontend

> Pendiente de implementar. Stack por definir.
